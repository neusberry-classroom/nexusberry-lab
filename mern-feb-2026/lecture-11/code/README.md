# Employee Profile Generator
Lecture 11: Functions, Closures & Higher-Order Array Methods

## What You'll Build
An employee profile generator that uses functions, closures, and higher-order array methods to process, transform, filter, sort, and analyze employee data — the foundational patterns for every data-driven application.

## How to Run
1. Open `index.html` in a browser
2. Open DevTools Console (`F12` → Console tab) to see all output
3. Or run directly with Node.js: `node script.js`

## File Structure
```
code/
├── README.md           ← This file
├── index.html          ← Browser entry point (links to script.js)
├── style.css           ← Minimal styling for the page
├── script.js           ← All lecture code organized by Parts
└── script-debug.js     ← Debug version with intentional bugs for live debugging
```

## Key Concepts Covered
- **Part 1:** Function declarations, expressions, arrow functions, default & rest parameters
- **Part 2:** Scope (global, function, block), scope chain, closures, pure functions
- **Part 3:** Callbacks, forEach, map, filter, find/findIndex
- **Part 4:** some, every, reduce, sort (with copy!), method chaining pipelines
- **Part 5:** TypeScript function types, return types, optional params, generics (preview)
